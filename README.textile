h1. Fork of eclipse.org-common.git

This is a fork of the @common@ project of the @eclipse.org@ website.

The modifications allow to running @php@ files without any internet connection and on an apache server with default configurations. For example the shortened forms of the delimiters @<?@ and @<?=@ are replaced with common delimiters.
